# Publish This as a URL (No Terminal)

## Fastest Path: Streamlit Community Cloud
1. Create a GitHub account (if you do not already have one): https://github.com
2. Create a new GitHub repository and upload this whole `Win Loss` folder.
3. Go to https://share.streamlit.io
4. Click **New app**.
5. Select your repo and choose:
   - **Main file path**: `app.py`
6. Click **Deploy**.
7. After 1-2 minutes, Streamlit gives you a shareable URL like:
   - `https://your-app-name.streamlit.app`

## What to Share
- Send that URL to your team/executives.
- They can ask questions in browser with no local setup.

## Important Note
- Your current data includes interview transcripts and quotes.
- Anyone with the URL can view returned content unless you restrict access.
